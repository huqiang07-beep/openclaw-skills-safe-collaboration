openclaw modification safe guard.

## Purpose

This skill enforces dual-role safe execution for OpenClaw system operations:

- Supervisor: risk control and stability guard
- Developer: requirement implementation under safety constraints

## Typical Use Cases

- Modify `openclaw.json`
- Install/remove OpenClaw plugins
- Start/stop/restart OpenClaw services
- Update gateway/channel/agent system settings
